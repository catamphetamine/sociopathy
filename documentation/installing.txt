Установка на домашнем компьютере:

Ставим виртуальную машину (например, VMWare Workstation)
Заводим в ней рабочую систему (например, Ubuntu Server)

Заводим пользователя sociopathy

sudo adduser sociopathy
addgroup admin
nano /etc/sudoers
%admin ALL=(ALL) ALL
sudo adduser sociopathy admin

Входим под ним

Ставим git
Скачиваем "репозиторий" git://github.com/kuchumovn/sociopathy.git в папку "repository" пользователя sociopathy

git clone git://github.com/kuchumovn/sociopathy.git repository

Идём на http://nodejs.org/download/ и скачиваем исходники node.js
Устанавливаем node.js
(наверное, лучше из репозитория joyent, чем прямо из исходников, как я описал здесь)

mkdir ~/sociopathy
cd ~/sociopathy
wget http://nodejs.org/dist/v0.8.9/node-v0.8.9.tar.gz
tar xvf node-v0.8.9.tar.gz
rm node-v0.8.9.tar.gz
mv node-v0.8.9 node
cd node
./configure
sudo make
sudo make install
cd ..

может быть ещё: npm install socket.io

Устанавливаем NginX

sudo -s
nginx=stable # use nginx=development for latest development version
add-apt-repository ppa:nginx/$nginx
apt-get update 
apt-get install nginx

Устанавливаем ImageMagick

sudo apt-get install imagemagick

Устанавливаем MongoDB

sudo apt-key adv --keyserver keyserver.ubuntu.com --recv 7F0CEB10
sudo echo "deb http://downloads-distro.mongodb.org/repo/ubuntu-upstart dist 10gen" >> /etc/apt/sources.list
sudo apt-get update
sudo apt-get install mongodb-10gen

Это будет папка с базой данных, её нужно ежедневно бекапить

mkdir /home/sociopathy/database

Устанавливаем Redis

wget http://redis.googlecode.com/files/redis-2.4.17.tar.gz
tar xzf redis-2.4.17.tar.gz
rm redis-2.4.17.tar.gz
mv redis-2.4.17 redis
cd redis
make

создаём свою папку с настройками в папке configuration (наподобие home_windows)
там в файле enginex.conf меняем пути "root" на свои (2 штуки)
в configuration.coffee - тоже
в configuration.private.coffee - тоже

настраиваем NginX, чтобы он использовал enginex.conf

sudo nano /etc/nginx/nginx.conf
ищем там строчку: include /etc/nginx/sites-enabled/*;
дописываем под ней: include /home/sociopathy/repository/configuration/[название папки с вашими настройками]/enginex.conf

создаём свой запускальщик наподобие: если на винде, то запустить.bat, если на линуксе, то run.sh
запускаем запускальщик

* Если что-то не запустится - смотрите логи ошибок всего используемого здесь ПО (nginx, базы данных, прочее)

Заходим на http://[IP]:8081/управление
Жмём "Создать хранилище"
Идём на главную и входим: Дождь со Снегом / 123
Опять идём в управление
Жмём "Заполнить хранилище"
Теперь можно полазить по сайту

//Подключаем услугу "внешний IP"
